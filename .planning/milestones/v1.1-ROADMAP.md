# Milestone v1.1: cloud + distribution + harnesses

**Status:** ✅ SHIPPED 2026-06-01
**Phases:** 5-7
**Total Plans:** 19 (Phase 5: 8 · Phase 6: 5 · Phase 7: 6)

## Overview

v1.1 lifts v1.0's local substrate to real multi-host cloud runs and adds the algo-layer harness surface that v1.2 PPO/GRPO will consume via trait objects. Three phases: real AWS/GCP cloud adapters + object-store-backed streaming snapshots (Phase 5); a pull-based multi-node coordinator with work-stealing, storage-backed restart, spot-drain, and split-brain fencing (Phase 6); and three harness crates — text-completion env, sandboxed tool harness, bundled-eval harness — plus the top-level `rollout eval` CLI (Phase 7). 12/12 in-scope requirements satisfied; HARNESS-04 (eval gate) deferred to v1.2.

- Layered tool-sandbox defense (rustix namespaces + landlock + seccompiler + cap-std + cgroups v2), seccomp installed last; `require_landlock=true` fail-closed on kernel <5.13; macOS = compile-only dev stub (process-isolated, explicitly NOT a VM/security perimeter).

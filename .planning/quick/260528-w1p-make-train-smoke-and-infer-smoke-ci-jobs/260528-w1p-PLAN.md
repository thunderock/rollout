<objective>
Make the `infer-smoke` and `train-smoke` GitHub Actions jobs always-on and free on standard public runners.

Today both jobs are gated behind `if: vars.ROLLOUT_*_AVAILABLE == '1'` and the underlying scripts self-skip on the same env vars, so they never fire on public CI. `infer-smoke` also installs the ~10GB CUDA `vllm` wheel which requires a GPU. The vLLM glue already selects CPU automatically via a `torch.cuda.is_available()` probe (see `python/rollout/backends/vllm/engine.py`), so no backend code changes are needed.

Purpose: Give every PR a real end-to-end inference + SFT smoke (BACKEND-02 / Phase-4 exit criteria) at zero cost on the free 4-vCPU/16GB ubuntu-latest public runners.
Output: Two ungated CI jobs running CPU wheels with pip + HuggingFace caching, two scripts with the self-skip removed, and a refreshed CPU-mode doc.
</objective>
